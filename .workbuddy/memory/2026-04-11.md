# 2026-04-11

## GeckoView Browser mod 修复
- 将 manifest.json 中 URL entity 的 type 从 `text_sensor` 改为 `text`
  - `text_sensor` 在 HA 里生成 `sensor.xxx`，用户需要的是 `text.xxx`
  - 修改了 sources/ 和 mods/ 两个目录下的 manifest.json
- 修复 putenv() UnsatisfiedLinkError：libmozglue.so 未加载导致 native 方法找不到实现
  - 在调用 GeckoLoader.putenv() 前先 System.load libmozglue.so
- 构建 JAR 需要 Java 17（d8 要求 class file version 55+），JAVA_HOME 设为 JBR 17
